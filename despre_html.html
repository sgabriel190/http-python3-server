Limbajul HTML

HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html îl găsiţi în imaginea următoare care se află la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.
Imaginea are atributul height de valoare 147.


Tag-urile html cele mai utilizate în realizarea tabelelor sunt:


table defineste un tabel
tr un rand într-un tabel
td o celula într-un tabel

Observaţie: Pentru a pune un contur unui tabel se poate (dar nu se recomandă) să se pună atributul border="1".
La validare va da eroare: The border attribute on the table element is obsolete.


Formularele se pot crea cu ajutorul tag-ului form care poate conţine tag-urile: input, textarea, label, select, option, button.


Pentru mai multe detalii consultaţi cursul sau tutorialul HTML Forms and Inputs de la adresa http://www.w3schools.com/html/html_forms.asp

Un exemplu de formular este urmatorul:
Nume:
Prenume:
Parolă:
Culoarea preferată:
Descriere:
Sunt de acord cu conditiile



Pentru a crea liste se pot folosi tag-urile:


  • ul - listă neordonată
  • ol - listă ordonată
  • li - item din listă
  • dl - listă cu descrieri
  • dt - termen dintr-o lista cu descrieri
  • dd - descrierea dintr-o listă cu descrieri

Pentru a insera un document HTML în altul se poate folosi tag-ul <iframe>.
Astfel, exemplul de mai jos inserează conţinutul de la adresa https://html.spec.whatwg.org/multipage/.
